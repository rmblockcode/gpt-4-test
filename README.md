# GPT-4 Test

Proyecto file downloader de videos de Youtube. Proyecto educativo para probar GPT-4

Para ejecutar la api de flask:

    python3 -m venv venv

    source venv/bin/activate

    pip install -r requirements.txt

    python bot.py
